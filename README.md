# 📉 Telecom KPI Dashboard Engine

![Python](https://img.shields.io/badge/Python-3.10-blue) ![Matplotlib](https://img.shields.io/badge/Matplotlib-Charts-orange) ![Excel](https://img.shields.io/badge/Excel-Reports-green)

## Overview
A Python-based KPI computation and dashboard engine for telecom operations. Generates multi-chart dashboards and Excel summary reports from raw operational data — designed to mirror the kind of management reporting built at IHS Towers.

## Dashboards Generated
1. **Executive Summary** — top-level KPIs across the portfolio
2. **Site Performance** — individual site scores and availability trends
3. **Vendor Analysis** — delivery performance by vendor
4. **Supply Risk** — sites approaching critical fuel levels

## How to Run
```bash
pip install -r requirements.txt
python dashboard/generate_dashboard.py
```
